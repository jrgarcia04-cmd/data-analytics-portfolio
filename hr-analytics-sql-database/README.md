# HR Analytics SQL Database Project (MySQL)

## Project Overview
This project involved designing and implementing a relational database to analyze HR and employee data. The objective was to structure data efficiently and use SQL queries to generate meaningful analytical insights.

---

## Objectives
- Design a normalized relational database schema
- Implement primary and foreign key relationships
- Store and manage HR-related employee data
- Develop analytical SQL queries for reporting

---

## Database Design
The database includes structured tables representing:
- Employees
- Departments
- Positions
- Performance metrics
- Other HR-related attributes

Relationships were defined using primary and foreign keys to maintain data integrity.

---

## SQL Techniques Used
- SELECT statements
- WHERE filtering
- INNER JOIN and LEFT JOIN
- GROUP BY and aggregation functions (COUNT, SUM, AVG)
- Subqueries
- Data constraints

---

## Key Skills Demonstrated
- Relational database design
- Data normalization
- Writing complex SQL queries
- Analytical reporting
- Data integrity enforcement

---

## Tools Used
- MySQL
- SQL scripting

---

## How to Run
1. Open the SQL script in MySQL Workbench.
2. Execute the database creation script.
3. Run analytical queries to generate reports.

---

## Outcome
This project demonstrates the ability to design structured databases and extract actionable insights using SQL, supporting data-driven HR decision-making.
