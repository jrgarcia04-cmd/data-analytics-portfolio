# Sales Data Exploratory Data Analysis (R)

## Project Overview
This project analyzes electronic product sales data using R to identify revenue trends, product performance, and key business insights. The goal was to explore patterns in sales performance and support data-driven decision-making.

---

## Objectives
- Clean and prepare raw sales data
- Perform exploratory data analysis (EDA)
- Identify trends in revenue and product categories
- Generate visualizations to support findings

---

## Methods & Approach
1. Data cleaning and preprocessing  
2. Summary statistics and aggregation  
3. Revenue trend analysis  
4. Product-level performance comparison  
5. Data visualization using ggplot2  

---

## Tools & Libraries
- R
- tidyverse
- dplyr
- ggplot2

---

## Key Skills Demonstrated
- Data cleaning and transformation
- Exploratory data analysis
- Data visualization
- Analytical thinking
- Interpreting business metrics

---

## How to Run
1. Open `Sales_data_for_electronic_products.R` in RStudio.
2. Install required packages if needed.
3. Run the script from top to bottom.

---

## Outcome
The analysis highlights revenue patterns, identifies high-performing products, and demonstrates practical application of R for business analytics.
